If you want to link a specific collection use the attributes:

> shopify=collection
>
> el=collection-link
>
> udesly-data=[ your collection handle]

To create a link to all the products of your store, you have to set the udesly-data value to **all**:

> shopify=collection
>
> el=collection-link
>
> udesly-data=all
